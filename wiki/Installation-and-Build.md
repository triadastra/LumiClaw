# Installation and Build

## Requirements

- macOS 14+
- Xcode / Swift toolchain
- API keys for cloud providers you plan to use

## Build

```bash
git clone https://github.com/Lumicake/LumiClaw.git
cd LumiClaw
swift build
```

## Run

```bash
./run_app.sh
```

Or run from Xcode with the `LumiAgent` scheme.

## API Keys

In app: `Settings -> API Keys`

- OpenAI
- Anthropic
- Gemini
- Brave (optional)
- Ollama URL (local)
